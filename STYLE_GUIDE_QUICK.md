# Chain Gang Book — Quick Style Guide

Use these patterns consistently in `book.md`.

---

## Chapters
```markdown
# Chapter X — Title Here
```

## Sidebars
```markdown
### SIDEBAR: From the Sideline — Title
[Text]
<!-- end-sidebar -->
```

## Pro Tips
```markdown
**Pro Tip:** Text
<!-- end-protip -->
```

## Figures
```markdown
![Alt text](images/file.jpg)
*Figure X: Caption text.*
```

## Appendices
```markdown
## Appendix A — Title
```

## Checklists
```markdown
- [ ] Item one
- [ ] Item two
```

## Other Sections
```markdown
# Preface
# Acknowledgements
# Conclusion
```
